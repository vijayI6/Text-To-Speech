# Text to Speech Converter

Text to Speech Converter is a web-based tool that allows users to input text and instantly hear its audio representation. This implementation uses HTML, CSS, and JavaScript, integrating modern browser APIs for seamless text-to-speech conversion.

## Table of Contents
- [Demo](#demo)
- [Features](#features)

## Demo
You can view a live demo of the tool at voicesynthesis.netlify.app

## Features
- Real-time text to speech conversion
- Adjustable speech settings (voice, speed, pitch)
- Simple and intuitive user interface
- Supports multiple languages and voices (where available)
- Copy the spoken text to clipboard functionality

## Usage
1. Open the tool in your web browser.
2. Type or paste the text you wish to convert into the input field.
3. Select your preferred voice, adjust speed and pitch as desired.
4. Click the "Listen" button to hear the text spoken aloud.
5. Optionally, use the "Copy Text" button to copy the input text to your clipboard.
